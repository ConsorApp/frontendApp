## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```


### Templating/components

Uso [shadcn](https://ui.shadcn.com/) para tener components lindos y todo el estilo.
para agregar un componente:

`pnpm dlx shadcn@latest add button`

Por ejemplo para agregar el button.
